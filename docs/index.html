Installation Guide

Step-by-step installation via Terminal

  1. ⌘

    Open Terminal

    Press ⌘ + Space to open Spotlight, type Terminal and press Enter.

  2. 2

    Install ClipMind

    Paste this command and press Enter. Homebrew will download and install the app; you may be prompted for your Mac password.

    Terminal — Install ClipMind
    $brew install --cask clipmind
  3. ✓

    Done

    When the command finishes, open ClipMind from Applications or Spotlight (⌘ + Space).

About this app

ClipMind is a fast, privacy-first clipboard manager for macOS with built-in AI — history, search, rewrite, summarize, translate, and grammar fixes from the menu bar.

Clipboard history

Nothing you copy is lost; everything stays searchable.

AI actions

Rewrite, summarize, translate, fix grammar without switching apps.

Privacy

Local-first storage; no tracking in the product vision.
