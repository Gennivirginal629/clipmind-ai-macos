Fast, privacy-first clipboard manager for macOS with built-in AI — unlimited history, instant search, rewrite, summarize, translate, and grammar fixes from the menu bar.

## Features

- **Unlimited clipboard history** — everything you copy stays searchable, forever
- **Instant search** — find any past snippet in milliseconds
- **AI rewrite** — rephrase any text in a different tone or style
- **Summarize** — condense long pastes into a single sentence
- **Translate** — translate clipboard content to any language instantly
- **Grammar fixes** — clean up writing without opening another app
- **Menu bar first** — always one click away, never in your way
- **Privacy by design** — all data stored locally, no cloud sync, no tracking
